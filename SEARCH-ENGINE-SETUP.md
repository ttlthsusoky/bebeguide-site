## 🔵 Naver Search Advisor (네이버 서치어드바이저)

**중요도**: ⭐⭐⭐⭐⭐ (한국 검색 시장 점유율 60%+)

### 등록 방법

**1단계: 네이버 서치어드바이저 가입**
- URL: https://searchadvisor.naver.com
- 네이버 계정으로 로그인

**2단계: 사이트 등록**
1. "웹마스터 도구" 선택
2. "사이트 등록" 클릭
3. 사이트 URL 입력: `https://be-be-guide.com`

**3단계: 소유권 확인**

**방법 1: HTML 파일 업로드 (추천)**
```
1. 서치어드바이저에서 제공하는 HTML 파일 다운로드
   예: naver1234567890abc.html
2. 파일을 website 루트 디렉토리에 업로드
3. https://be-be-guide.com/naver1234567890abc.html 접속 확인
4. "소유확인" 버튼 클릭
```

**방법 2: HTML 메타태그**
```html
<!-- index.html의 <head> 안에 추가 -->
<meta name="naver-site-verification" content="제공받은코드"/>
```

**4단계: 사이트맵 제출**
- "요청" > "사이트맵 제출"
- URL 입력: `https://be-be-guide.com/sitemap.xml`
- RSS 추가: `https://be-be-guide.com/feed.xml`

**5단계: 수집 요청**
- "요청" > "URL 수집 요청"
- 주요 페이지 URL 입력 (일 10개 제한)

**기대 효과**: 1-2주 후 네이버 검색 노출 시작

1. "사이트 추가" 클릭

**4단계: Google Search Console에서 가져오기**
- "Google Search Console에서 가져오기" 선택
- Google 계정으로 인증
- 자동으로 사이트맵 및 설정 동기화

## 🎥 YouTube SEO (유튜브 검색 최적화)

**중요도**: ⭐⭐⭐⭐⭐ (세계 2위 검색엔진, 구글 다음)

### 유튜브가 중요한 이유
- 전 세계 2위 검색엔진 (월간 사용자 20억+)
- 육아 콘텐츠 검색 비중 매우 높음
- "신생아 준비물", "이유식 만들기" 등 영상 검색 선호
- 구글 검색 결과에 유튜브 영상 우선 노출

### YouTube Studio 설정

**1단계: 채널 개설**
- URL: https://studio.youtube.com
- 구글 계정으로 로그인
- "채널 만들기" → "베베가이드" 입력

**2단계: 채널 최적화**
```
채널명: 베베가이드 - 월령별 육아 가이드
채널 설명:
0~36개월 월령별 육아 체크리스트, 이유식 레시피,
신생아 돌보기, 예방접종 가이드까지!
초보 부모를 위한 실전 육아 꿀팁을 영상으로 만나보세요.

키워드: 육아, 신생아, 이유식, 월령별 체크리스트,
출산준비물, 예방접종, 아기발달, 육아팁
```

**3단계: 웹사이트 연결**
- "맞춤 설정" > "기본 정보"
- 링크 추가: https://be-be-guide.com
- 소셜 미디어 링크 추가

**4단계: 채널 ID 확인**
- "설정" > "채널" > "고급 설정"
- 채널 ID 복사 (예: UCxxxxxxxxxxxxxxxxxxx)

### 웹사이트에 유튜브 연동하기

**index.html에 추가할 코드**:
```html
<!-- YouTube 채널 링크 -->
<link rel="alternate" type="application/rss+xml"
      title="베베가이드 YouTube"
      href="https://www.youtube.com/feeds/videos.xml?channel_id=YOUR_CHANNEL_ID">

<!-- Schema.org 마크업 (Organization) -->
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "베베가이드",
  "url": "https://be-be-guide.com",
  "logo": "https://be-be-guide.com/images/og-bebeguide.webp",
  "sameAs": [
    "https://www.youtube.com/@bebeguide"
  ]
}
</script>
```

### 유튜브 SEO 최적화 방법

**1. 제목 작성법**
```

🛒 쿠팡 할인가 보기: [쿠팡 링크]

#신생아준비물 #출산준비 #베베가이드
```

### 기대 효과

✅ **검색 노출 증가**
- 유튜브 검색 1위 = 구글 검색에도 영상 노출
- "신생아 준비물" 검색 시 상위 노출 가능

✅ **웹사이트 트래픽 증가**
- 유튜브 → 베베가이드 유입
- 월 1,000~5,000명 유입 가능 (채널 성장 시)

✅ **브랜드 인지도 상승**
- 영상으로 신뢰도 구축
- 얼굴 공개 시 더 높은 신뢰도

✅ **쿠팡 제휴 수익 증가**
- 영상에서 제품 소개 → 구매 전환율 높음

## 📞 문제 해결

### 사이트가 검색되지 않아요
- 등록 후 2-4주 대기 필요

- 수동 URL 수집 요청

- 콘텐츠 품질 개선
- 키워드 최적화
- 외부 링크 확보 (백링크)
