========================================
베베가이드 네이버 블로그 완벽 시작 가이드
========================================

📌 목차
1. 네이버 블로그 개설 방법 (5분)
2. 기본 설정 방법
3. 포스팅 올리는 방법 (단계별)
4. SEO 최적화 팁
5. 준비된 10개 글 정보


========================================
STEP 1: 네이버 블로그 개설 (5분)
========================================

1. 네이버 접속 (naver.com)
2. 로그인 (네이버 계정 필요)
3. 상단 검색창 옆 "블로그" 클릭
4. "내 블로그" 클릭
5. 블로그 주소 설정
   - 추천: bebeguide 또는 bebe-guide
   - 한글 이름: 베베가이드
6. "블로그 개설하기" 클릭

✅ 완료!


========================================
STEP 2: 기본 설정 (5분)
========================================

[관리 → 기본설정]

1. 블로그 제목: 베베가이드
2. 블로그 설명: 초보 부모를 위한 육아 완벽 가이드 | 수유·건강·발달·안전
3. 블로그 공개 설정: 전체 공개
4. 검색 허용: 허용 (중요!)

[관리 → 카테고리 설정]

다음 카테고리 만들기:
- 📍 전체보기 (기본)
- 🍼 수유·이유식
- 💊 건강·질병
- 👶 성장·발달
- 🛁 돌봄·관리
- 🛡️ 안전
- 🛒 육아용품

✅ 완료!


========================================
STEP 3: 포스팅 올리는 방법
========================================

1. "글쓰기" 버튼 클릭
2. 제목 입력
3. 본문 입력 (제가 준비한 텍스트 복사-붙여넣기)
4. 카테고리 선택
5. 태그 입력 (쉼표로 구분)
6. 공개 설정: 전체공개
7. "발행" 클릭

⏱️ 글 1개당 2-3분 소요
⏱️ 10개 전체: 약 30분


========================================
STEP 4: SEO 최적화 팁
========================================

✅ 네이버가 자동으로 해주는 것:
- 네이버 검색 노출
- 모바일 최적화
- 방문자 통계

📝 사용자가 하면 좋은 것:
1. 제목에 검색 키워드 포함
   예: "아기 열 대처법" → "아기 열·감기 대처법 완벽 가이드 | 해열제 용량"

2. 태그 5-10개 입력
   예: 아기열, 감기, 해열제, 타이레놀, 육아, 신생아

3. 첫 문단에 핵심 키워드 포함

4. 정기적으로 포스팅 (주 1-2회 이상)

5. 이미지 추가 (선택사항)
   - 무료 이미지: unsplash.com, pixabay.com

========================================

준비된 블로그 글 (카테고리별):
